##  Objetivo general

Explorar y usar de forma práctica distintas plataformas de IA (modelos generativos, copilotos de código, RAG, etc.), evaluando:

- Capacidades y limitaciones.
- Calidad de las respuestas/resultados.
- Posibles aplicaciones educativas.
- Integración en flujos de trabajo reales (APIs, notebooks, repositorios).

El laboratorio culmina con un **caso integrador** donde se combinan varias herramientas para resolver un problema real en educación.

## Reflexión y uso de IA

Durante el laboratorio se utilizaron varias IAs (ChatGPT, Gemini, Copilot, Codeium, Claude, etc.) como apoyo para:

Generar código y prompts.
Mejorar redacción de informes.
Analizar y refactorizar código existente.
Todas las salidas de IA fueron revisadas, ajustadas y validadas manualmente, en línea con las políticas de integridad académica.
